# 🍎 Think_Different 🍎

## 🔗 Demo
http://43.201.67.221:8080

## 📌 프로젝트 소개

Think_Different는 커플의 일정, 데이트 비용, 데이트 기록을 함께 관리할 수 있는  
Spring Boot 기반 커플 전용 웹 서비스입니다.

회원 인증(Spring Security)을 기반으로 사용자를 관리하고,  
초대코드를 통해 커플을 연결한 뒤 두 사용자가 하나의 커플 공간에서  
일정 공유, 데이트 비용 관리, 데이트 기록 관리를 할 수 있도록 설계했습니다.

백엔드 구조 설계, JPA 엔티티 모델링, 연관관계 매핑, 인증 처리,  
DB 연동(JPA/MySQL), Docker 기반 배포 환경 구성을 중심으로 개발했습니다.

## 🚀 주요 기능

### 🔐 회원 인증 (Spring Security)
- 회원가입 / 로그인 / 로그아웃
- 인증 기반 접근 제어
- 로그인 사용자 기준 데이터 처리

### 💑 커플 연결
- 초대코드 생성
- 초대코드 입력을 통한 커플 연결
- 이미 연결된 사용자 재연결 방지
- 커플 단위 데이터 관리 구조 설계

### 📅 일정 공유
- 커플 일정 등록 / 조회 / 수정 / 삭제
- 데이트 날짜, 시간, 장소, 메모 관리
- 커플 기준 일정 조회

### 💰 데이트 비용 관리
- 데이트 비용 등록 / 조회 / 수정 / 삭제
- 지출 금액, 결제자, 사용 내역 관리
- 커플 단위 데이트 비용 기록

### 📝 데이트 기록
- 데이트 기록 등록 / 조회 / 수정 / 삭제
- 데이트 장소, 날짜, 메모 관리
- 일정 이후 추억을 기록할 수 있는 구조 설계

## 📂 주요 도메인 구조

- Member: 사용자 정보 및 인증 기준 엔티티
- Couple: 연결된 커플 관계 관리
- InviteCode: 커플 연결을 위한 초대코드 관리
- Schedule: 커플 일정 관리
- Expense: 데이트 비용 관리
- DateRecord: 데이트 기록 관리

## 🧩 프로젝트 방향

기존 개인 가계부 중심 서비스에서 확장하여,  
커플이 함께 사용하는 데이트 관리 서비스로 기획을 변경했습니다.

단순 지출 기록이 아니라 커플의 데이트 전후 흐름을 관리하는 것을 목표로 합니다.

- 데이트 전: 일정 공유
- 데이트 중/후: 비용 기록
- 데이트 후: 추억 기록

이를 통해 커플이 하나의 공간에서 데이트 계획, 비용, 기록을 통합 관리할 수 있도록 구현하고 있습니다.
